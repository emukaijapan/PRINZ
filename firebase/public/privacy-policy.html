プライバシーポリシー

最終更新日: 2026年2月6日

PRINZ運営（以下「当運営」）は、PRINZアプリ（以下「本アプリ」）におけるユーザーの個人情報の取り扱いについて、以下のとおりプライバシーポリシーを定めます。

1. 収集する情報

1.1 ユーザーが提供する情報

  • 認証情報: Apple Sign-Inを通じて取得するユーザーID
  • ユーザー属性: 性別、年代、パーソナルタイプ（10種類の性格タイプ）
  • チャットテキスト: スクリーンショットから抽出したテキスト、または直接入力されたテキスト

1.2 自動的に収集される情報

  • 利用状況: 1日あたりの返信生成回数
  • 購入情報: サブスクリプションの状態（RevenueCat経由）
重要: 画像はサーバーに送信されません
スクリーンショットの文字認識（OCR）処理はすべてお使いのデバイス上で行われます。画像自体が当社サーバーに送信されることはありません。

2. 情報の利用目的

収集した情報は、以下の目的で利用します。

  • AIによる返信案の生成（ユーザー属性に基づくパーソナライズ）
  • サブスクリプションの管理
  • 利用回数の制限管理（Free: 5回/日、Premium: 100回/日）
  • サービスの改善・不正利用の防止

3. 第三者への情報提供

本アプリでは、以下の第三者サービスを利用しています。

サービス 提供元 共有する情報 目的
Firebase Google LLC ユーザーID、利用回数 認証、データベース
OpenAI API OpenAI, Inc. チャットテキスト、ユーザー属性 AI返信生成
RevenueCat RevenueCat, Inc. 購入情報 サブスクリプション管理

各サービスのプライバシーポリシー:

  • Firebase プライバシーポリシー
  • OpenAI プライバシーポリシー
  • RevenueCat プライバシーポリシー

4. データの保存と保持期間

4.1 デバイス内保存

  • 返信履歴: 最新30件までローカルに保存。アプリ削除時に消去されます。
  • ユーザー設定: デバイス内に保存。アプリ削除時に消去されます。

4.2 クラウド保存

  • 利用回数データ: 日次でリセットされます。
  • ユーザー情報: アカウント削除まで保持されます。

5. ユーザーの権利

ユーザーは以下の権利を有します。

  • アクセス権: ご自身のデータへのアクセスを要求できます。
  • 削除権: アカウントおよびデータの削除を要求できます。
  • オプトアウト: サブスクリプションはいつでも解約できます。

データの削除をご希望の場合は、下記のお問い合わせ先までご連絡ください。

6. セキュリティ

当運営は、お客様の情報を保護するために以下の対策を講じています。

  • Firebase Authentication による認証
  • Firebase Security Rules によるアクセス制御
  • APIキーのサーバーサイド秘匿
  • HTTPS による通信の暗号化

7. 子どものプライバシー

本アプリは17歳以上のユーザーを対象としています。17歳未満の方は本アプリをご利用いただけません。

8. プライバシーポリシーの変更

当運営は、必要に応じて本ポリシーを変更することがあります。重要な変更がある場合は、アプリ内または本ページにて通知します。変更後も本アプリを継続してご利用いただくことで、変更後のポリシーに同意したものとみなします。
